## Features

- No runtime dependencies
- Abstracts common used FiveM practices
- Entity management through class objects (i.e. `Vehicle` and `Ped` entities)
- Server and Client side variants on wrapper

## Download & Install

`pnpm add @nativewrappers/fivem`
